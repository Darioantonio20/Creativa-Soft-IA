# Creativa-Soft-IA

Creativa Soft-IA is a technology company focused on designing and developing intelligent digital solutions that help organizations innovate and scale.

## Project Structure

### frontend/
React.js application with Tailwind CSS using Atomic Design principles.

The frontend is a modern, responsive web UI/UX with:
- **React.js** - Component-based UI framework
- **Tailwind CSS** - Utility-first CSS framework
- **TypeScript** - Type-safe JavaScript
- **Atomic Design** - Component architecture methodology

See [frontend/README.md](frontend/README.md) for detailed documentation.

## Getting Started

### Frontend

```bash
cd frontend
npm install
npm start
```

The application will be available at [http://localhost:3000](http://localhost:3000).

## Technology Stack

- **Frontend**: React.js, TypeScript, Tailwind CSS
- **Architecture**: Atomic Design Pattern
- **Build Tool**: Create React App
